# Parina Framework Manifiesto  
## *Altiplano Edition — Kawkhantix juk'akiw juk'ampïki*

Parinaqa janiw mä taqpach llikachakiti (full-stack).  
Janiw ukhamäñ munkiti.  
Janiw taqi kuns lurañ atipkiti.

Amtäwipax juk'amp manq'ankiwa:

> **Luririrux ina ch'ama ch'axwawinakat qispita lurayañataki.  
> Wasitat amuyt'añataki. Wasitat uñjañataki. Amtampi lurañataki.**

---

# 1. Altiplano taypinkir amukt'awi

Mä uraqin kawkhantix wali ch'axwawikiwa —jach'a llikachanaka, imantat patanaka, lurat qillqanaka, ch'ama nayra layqawinaka—  
Parinaqa mä mayja amta uñt'ayi:

**sinti phiskhu lurawi mä ch'axwawi uñt'ata.**

Altiplanox amtapuni ch'usa uñt'ayatawa.  
Ukatpachas wali k'achakiwa.

---

# 2. Khusa taypi churawi mä lurayawitaki, janiw llawuntatäkiti

Parinaqa wakiskiriki churayi:

- mä nayriri p'iqiñchiri (front controller)  
- mä qhananchat thakhi churi (router)  
- mä taypi khitiri (kernel)  
- sapaqt'at luririnaka (handlers)  
- juk'a uñtawipa (views)  
- qhananchat kutt'ayawinaka (responses)  

Janiw ch'aman layqa uñch'ukiwinakäkiti.  
Janiw imantat mayjachäwinakäkiti.  
Janiw jan amuykir conventions-nakäkiti.

> **Juk'amp munkaxa, jumapachaw lurt'añama.  
> Ukhamatwa qillqatamat jach'anchatäta.**

---

# 3. Ch'amampi lurawix ampararuw kutt'ani

Jach'a llikachanakax taqi kuns lurt'apxi ukat janiw amuyt'añ thakh churapktamti.  
Parina Framework janiw ukhamäkiti.

Akanxa, luririx:

- base de datos ukaru mantañ amti  
- pachpa arunakar uñt'ayañ amti  
- sapa uñachayirinak phakhinakar uñt'ayañ amti  
- thakhip jilachayañ amti  
- kawkiri patanakarux wakisi uk amti  

Ukhamatwa aka framework-ax janiw wawatak uñjktamti.  
Jumarux wali jach'anchtamwa.

---

# 4. Taqi kawkharutix jan utjki... amtapuniwa

Parina Framework janiw utjkiti:

- ORM  
- migrations  
- queues (suyirinaka)  
- caching  
- ch'ama templates  
- magic helpers  
- magic/runtime auto–scaffolding (scaffolding lurawix CLI thakhi ukankiwa)  
- phaxsi thakhinaka  

¿Ukhax mä pantawiti?

Janiw khusäkiti.  
Mä amtayawiwa: **jumax lurt'asmawa kawkharutix wakiski ukpacha**.

---

# 5. Qillqatax p'iqimar katuqañapawa

Mä llikachatix jan amuykmaxa  
janiw jumax p'iqinchktaati.

Parina uñch'ukiwipax ukham lurt'atawa:

- mä ch'eqa t'aqa qillqaru (whiteboard),  
- mä pichaw silluru (napkin),  
- amuyumaru.

Mä lurawi sinti ch'amächi 30 saniñan qhananchañataki…  
janiw akankañapäkiti.

---

# 6. Jach'achawi mä sapa mayja lurataki

Parina Framework janiw taqi kuns lurat churktamti.  
Mä ch'usa lienzo churtam, samichirinaka, ukat sapa lurat thakhi.

K'achachirix jumawa.

---

# 7. Qhipa arsuwi

> **"Janiw mä llikachak churksma jank'a qillqt'añataki.  
> Mä llikachak chursma khusa qillqt'añataki."**

Juk'a qillqata.  
Aswan qhananchawi.  
Aswan amtäwi.  
Aswan p'iqiñchawi.

Uka taypin Parina uñt'ata.  
Altiplano amukt'awipa:  
jach'a patanakana, phiskhu lurawina, amukt'awina…  
ukat k'achacht'awi kawkharutix wakiski ukapacha.

---

**Amtäwix tukusiwa.**
